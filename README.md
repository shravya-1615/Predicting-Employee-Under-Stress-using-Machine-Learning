# Employee Stress Prediction using Machine Learning

This project predicts employee stress levels using machine learning techniques. 
It includes data preprocessing, feature selection, and model training using LightGBM.

## Features
- Data preprocessing
- Feature importance analysis
- Machine learning model
- Performance evaluation

## Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- LightGBM

## How to Run
pip install -r requirements.txt
python app.py
